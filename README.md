# Web UI proof of concept

A proof of concept of a react web ui for a C++ app.

## Getting Started

    ./build
    _build/webui frontend/build

Then open `http://localhost:3003` in a browser.

## UI development

    ./build
    _build/webui &
    cd frontend && npm start

## Deployment

    ./install
    cd _install && bin/webui
